# Анализатор сетевого трафика

Программа для обнаружения аномалий в сетевом трафике методом статистического анализа.

## Возможности

- Сравнение двух наборов данных (baseline и current)
- Поиск новых устройств в сети
- Поиск пропавших устройств
- Обнаружение аномальной активности (правило трёх сигм)
- Сохранение и загрузка состояния анализа
- Генерация аналитических отчётов в JSON
